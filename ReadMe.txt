Project dRainbow

I dette prosjektet skal vi lage en hjemmeside der brukeren skal 
kunne velge blant 3 kategorier av bilder, 
3 kategorier av lyd og 3 kategorier av tekst, 
og basert på disse valgene få generert en utstilling
 med minimum 4 kombinasjoner av et bilde, en tekst og en lyd


Built With

React 
Ajax
Node.js
